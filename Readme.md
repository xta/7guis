## Tasks

1. Counter
2. Temperature Converter
3. Flight Booker
4. Timer
5. CRUD
6. Circle Drawer
7. Cells

Caveat Emptor. This repository is educational (learning SwiftUI) and does not demonstrate SwiftUI best practices.
